# Security Policy

This repository is documentation and scripts only.

Do not store:
- secrets, tokens, or API keys
- private SSH keys or credentials
- private project data

Inventory notes:
- docs/appendix/apt-sources.txt lists external repositories; treat it as public info.
- docs/appendix/* may include host-specific metadata; review before sharing.

If you find sensitive data here, please remove it and rotate credentials.
