# Политика безопасности

Эта репа содержит только документацию и скрипты.

Не храним:
- секреты, токены, API ключи
- приватные SSH ключи или креды
- приватные данные проектов

Про инвентарь:
- docs/appendix/apt-sources.txt содержит внешние репозитории; считай это публичной инфой.
- docs/appendix/* может включать метаданные машины; перед публикацией проверяй.

Если нашел чувствительные данные — удаляй и меняй доступы.
