# settings_ubuntu

This repository documents how to set up Ubuntu from zero to match the current machine state.

Two setup modes:
- Manual: copy/paste steps with explanations and why each dependency is needed.
- Semi/auto: baseline scripts with optional profiles for people who want a fast setup.
