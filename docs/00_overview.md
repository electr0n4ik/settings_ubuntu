# Overview

Bro, the goal is simple: reproduce this Ubuntu setup from zero, clean and predictable.
Target OS: Ubuntu 22.04.5 LTS (jammy).

What is covered:
- OS packages and repositories
- Desktop tweaks and GNOME Terminal profile
- Dev toolchains (Python, Go)
- Fast profiles for semi/auto setup

What is NOT covered:
- secrets, tokens, private keys
- project data or proprietary installers

Layout:
- docs/10_manual_setup.md: manual steps with copy/paste + why
- docs/20_auto_setup.md: baseline + optional profiles
- docs/40_gnome_terminal.md: terminal export/import
- docs/60_dev_env.md: Python and Go for beginners
- docs/90_audit.md: optional/unneeded review
- docs/30_apps.md: app groups and categories
- docs/scripts/: setup scripts
- docs/appendix/: raw inventory outputs

How to use:
1) Go manual with docs/10_manual_setup.md, or
2) Run the quick path in docs/20_auto_setup.md
