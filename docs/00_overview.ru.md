# Обзор

Бро, цель простая: воспроизвести эту Ubuntu с нуля быстро и понятно.

Что покрываем:
- пакеты и репозитории
- настройки рабочего стола и GNOME Terminal
- dev-среды (Python, Go)
- быстрые профили для полу/авто установки

Что НЕ покрываем:
- секреты, токены, приватные ключи
- данные проектов и закрытые инсталляторы

Структура:
- docs/10_manual_setup.ru.md: ручная настройка с пояснениями
- docs/20_auto_setup.ru.md: база + профили
- docs/40_gnome_terminal.ru.md: экспорт/импорт профиля терминала
- docs/60_dev_env.ru.md: Python и Go
- docs/90_audit.ru.md: ревизия лишнего
- docs/appendix/: сырые отчеты инвентаря

Как пользоваться:
1) читаешь docs/10_manual_setup.ru.md, или
2) идешь по быстрому пути из docs/20_auto_setup.ru.md
