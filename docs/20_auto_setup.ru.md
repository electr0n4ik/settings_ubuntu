# Полу/авто настройка

Бро, это быстрый путь: минималка + профили по вкусу.

Правила:
- База — безопасная и компактная.
- Профили добавляют только нужное.
- Скрипты повторяемые и с объяснениями.

## База (рекомендуется)
```
bash docs/scripts/auto_base.sh
```

Что делает:
- обновляет индексы
- ставит core утилиты
- добавляет пару десктоп удобств

## Профили
```
bash docs/scripts/profile_dev.sh
bash docs/scripts/profile_media.sh
bash docs/scripts/profile_office.sh
```

Каждый профиль поясняет, зачем эти пакеты.

Заметка:
- Профили не добавляют сторонние репо без документации.
