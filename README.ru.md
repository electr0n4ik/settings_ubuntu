# settings_ubuntu

Бро, это репа про настройку Ubuntu с нуля до состояния этой машины. Всё по-человечески и без духоты.

Два режима:
- Ручной: копипастим шаги и понимаем, зачем каждый пакет.
- Полу/авто: базовый скрипт + профили, чтобы быстро стартануть.
